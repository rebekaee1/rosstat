# Рабочий процесс по плану Forecast Economy v2

## Фазы: начал — доделал

- **Не начинать следующую фазу**, пока **текущая не закрыта** по критериям плана (вертикальный срез: что обещано в фазе — сделано и проверяемо).
- Если фаза большая — делить на **подзадачи внутри фазы**, но **не** считать фазу завершённой до полного критерия «готово» для этой фазы.

## Git и окружения

- **GitHub (`git push`)** — основной способ фиксировать прогресс; коммиты должны быть **согласованы** с тем, что реально сделано.
- **Прод-сервер (SSH / docker на 5.129.204.194 и т.п.)** — **не деплоить автоматически** и **не без явного запроса** владельца. Разработка и проверка — **локально** (и при необходимости CI), выкладка на сервер — **отдельным шагом**, когда скажете.
