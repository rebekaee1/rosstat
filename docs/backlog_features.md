## v3/edit_004 — рекламный блок в footer

**Тип:** идея, low priority.

**Контекст.** Пользователь упомянул возможность размещения рекламного блока в footer (партнёрские предложения по ипотеке/вкладам/брокерам).

**План.**
- Добавить компонент `<FooterAds />` в `Footer.jsx` (или создать footer, если его нет).
- Источник предложений: статический JSON `frontend/public/footer-ads.json` или эндпоинт `/api/v1/sponsors`.
